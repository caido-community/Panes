# Panes Plugin

Custom view modes for HTTP requests and responses in Caido.

## Overview

The Panes plugin lets you create custom transformations and view modes for HTTP requests and responses. Use Caido workflows or shell commands to transform data and view it directly in HTTP History, Replay, Sitemap, and other locations.

## Installation

```bash
pnpm install
pnpm build
# or during development
pnpm watch
```
